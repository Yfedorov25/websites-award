# Anti-AI-slop · ЖОРСТКИЙ банлист (українською) — фінальний фільтр копірайту
> Залізне правило: ЖОДЕН текст на наших сайтах не виходить, поки не пройде цей фільтр.
> Мета: копірайт має звучати як ЖИВА ЛЮДИНА-маркетолог рівня Федоріва, не як AI. Інтегрується в meaning-copy-system + мета-промпт як обов'язковий чек.

---

## ПУНКТУАЦІЯ ТА ТИПОГРАФІКА (технічні маркери AI)
- НІКОЛИ довге тире "—" (em-dash) у копірайті. AI зловживає ним. Замість: крапка, кома, двокрапка, дужки, або переформулювати. (Виняток лише якщо клієнт явно має його в гайдлайні.)
- Уникати парних тире-вставок "текст — пояснення — текст" (улюблена AI-конструкція).
- Не зловживати трикрапкою "..." для "загадковості".
- Не ставити емодзі в преміальний/luxury копірайт (доречні лише в грайливих consumer-брендах і то дозовано).
- Лапки українські "ялинки", не програмерські.

## AI-ЖАРГОН І КЛІШЕ (банлист фраз НІКОЛИ)
- "зануртеся у світ...", "занурення у...", "пориньте у..."
- "не просто X, а ціла філософія / це більше ніж X"
- "вражає своєю...", "вражаюча якість", "вражаючий досвід"
- "ми пропонуємо широкий спектр / асортимент"
- "команда професіоналів / досвідчених фахівців"
- "індивідуальний підхід до кожного клієнта"
- "якість, перевірена часом", "найкраще рішення для вас"
- "у світі, що постійно змінюється...", "у сучасному світі..."
- "відкрийте для себе", "дозвольте собі"
- "ми робимо все можливе", "ми прагнемо до досконалості"
- "синергія", "екосистема рішень", "комплексний підхід" (якщо не буквально)
- "вишуканий смак / вишукана елегантність" (порожні прикметники)
- "вже сьогодні" як заклик ("замовляйте вже сьогодні")
- "вашу увагу привертає", "не залишить байдужим"
- "налаштовані на результат", "орієнтовані на клієнта"

## РОСІЯНІЗМИ ТА КАЛЬКИ
- "співпадати" -> збігатися; "слідуючий" -> наступний; "на протязі" -> протягом / упродовж
- "приймати участь" -> брати участь; "вибачаюся" -> перепрошую / вибачте
- "задавати питання" -> ставити запитання; "по факту" -> насправді
- "в кінці кінців" -> зрештою; "не дивлячись на" -> попри / незважаючи на
- "являється" -> є; "відноситься" -> стосується / належить
- "оточуючі" -> довколишні; "міроприємство" -> захід
- "на даний момент" -> зараз / наразі; "в той же час" -> водночас

## КАНЦЕЛЯРИТ (мертва мова)
- "здійснюємо діяльність" -> працюємо / робимо
- "у разі виникнення необхідності" -> якщо треба
- "з метою забезпечення" -> щоб
- "надання послуг" -> послуги / допомагаємо
- "являє собою" -> це
- Пасив "вам буде надано" -> актив "ми дамо / отримаєте"
- Віддієслівні іменники: "здійснення доставки" -> "доставляємо"

## ПОЗИТИВНІ ПРАВИЛА (як писати ЖИВО)
1. Коротко: hero <= 6 слів, тіло <= 12 слів, одна думка = одне речення.
2. Дієслово дії на початку CTA: Замовляй / Розрахуй / Дізнайся / Спробуй / Почни.
3. Активний стан, не пасив. Конкретика, не абстракція (Harry Dry: уяви / спростуй).
4. Голос клієнта (VoC): реальні фрази з відгуків/інтерв'ю, не вигадані.
5. Образ замість терміна: "служба турботи" (не кол-центр), "жива статистика" (не аналітика).
6. Один фірмовий термін на бренд (Федорів: щоб конкурент НЕ міг повторити рядок).
7. Емоційний тригер у кожному ключовому рядку: турбота / гумор / гордість / простота / довіра / статус.
8. Докази, не обіцянки (Федорів): цифри, кейси, реальні фото замість прикметників.

## ФІНАЛЬНІ ТЕСТИ (перед публікацією кожного тексту)
1. Тест Федоріва (конкурент): "Чи міг би цей рядок написати будь-який конкурент?" Якщо так - slop, переписати, поки не зазвучить лише цим брендом.
2. Тест SMS: чи вкладається головна ідея в одне коротке речення/SMS?
3. Тест зубної щітки: чи продукт зрозумілий, потрібний, бажаний?
4. Тест em-dash: Ctrl+F "—" -> 0 збігів у фінальному копі.
5. Тест живої людини: прочитати вголос. Якщо жодна жива людина так не скаже - переписати.
6. Тест емоції: який із 6 тригерів несе рядок? Якщо нульовий - переписати.
