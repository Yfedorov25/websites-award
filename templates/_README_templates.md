# templates — Готові шаблони
> Копіюй і заповнюй. Економить час на старті кожного проєкту.

## Файли
- **claude-md-and-prompts.md** містить:
  - **CLAUDE.md шаблон** — кладеться в корінь репо проєкту (Claude Code читає автоматично): стек, правила, performance-бюджет, естетика.
  - **Шаблон ТЗ на секцію** — структура технічного завдання для кожної секції сайту (який recipe, ассети, копірайт, fallback).
  - **Майстер-промпти** — заміна ChatGPT для генерації start/end frame + animation prompts під Kling/Nano Banana Pro (3 стилі зі ScrollCraft methodology).

## Коли використовувати
- **Фаза 5 PIPELINE (ТЗ):** беремо CLAUDE.md шаблон + шаблон ТЗ.
- **Фаза 6 (білд), медіа-пайплайн:** майстер-промпти для генерації кадрів під Kling.
