# D_Novu — withnovu.com (РОЗБІР, TIER 1-2 voice/productivity app) ✅
> Awwwards HM. Voice-to-organize застосунок («talk your day»). Next.js + НАТИВНИЙ CSS scroll-driven (без Lenis/GSAP).
> ★ Цінний: award-плавність БЕЗ бібліотек (нативний CSS) + light-типографіка + voice-app копірайт.

## 1. СТЕК ★ (інша школа плавності):
- Next.js (Turbopack, CSS-modules) + **БЕЗ Lenis, БЕЗ GSAP**. Плавність на НАТИВНОМУ CSS: `animation-timeline: scroll()` (підтримується!) + 25 inline-transform елементів. Ймовірно scroll-driven CSS animations / View Transitions.
- 10.9 екранів, 7 секцій, 8 відео. ✅ TIER 1-2.
- ВИСНОВОК: award-вигляд досяжний БЕЗ важких бібліотек — на сучасному нативному CSS. Легше, швидше (нуль JS-бандла на анімацію).

## 2. ТОКЕНИ
- Типо: **Messina Sans weight 300** (H1 89px/lh1.0/ls-0.448 — light елегантний) + Instrument Serif + Victor Serif (антиква-акценти).
- Light weight 300 для великого H1 = вишукана легкість (як REF/Parallel). Mix гротеск+антиква.

## 10. ЗБАГАЧУЄ PB_scroll_smoothness (нативний CSS варіант) + voice-app:

- Light-display (weight 300) тренд для calm/premium.
- Privacy-секція знову (app-патерн).
