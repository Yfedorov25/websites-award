# ScrollCraft PRO — перебудована база (повний пакет)
> Операційна система створення award-winning cinematic / 3D / scroll-driven вебсайтів через Claude Code.
> Сенс веде техніку. Сайти, що закохують у бренд і приносять бізнесу гроші.

## ЯК ЗАЛИТИ В PROJECT
1. Видали ВСІ поточні файли в Project.
2. Заливай файли з цього архіву (можна зберегти структуру тек або залити плоско — Claude знайде по назвах).
3. Готово. Починай новий проєкт через **META_PROMPT.md** у новому чаті.

## З ЧОГО ПОЧАТИ
- **START_HERE.md** — точка входу, карта бази.
- **META_PROMPT.md** — двигун проєкту (скопіюй у новий чат).
- **_INDEX.md** — повний покажчик усіх 98 файлів.

## ЩО НОВОГО В ЦІЙ ВЕРСІЇ
- **META_PROMPT.md** — покроковий двигун: ніша → дослідження/скрап → сенси й копірайт за
  Федорівим → структура → візуалізація емоції рухом → інструменти/3D/Higgsfield MCP → ТЗ для Claude Code.
- **00_foundations/meaning-copy-system.md** — метод Федоріва (піраміда), UA-регістри, emotion→copy.
- **00_foundations/motion-system.md** — анатомія руху з виміру 29 студій: easing-бібліотека, темп, stagger, emotion→motion.
- **teardowns/** — 44 розбори (20 motion + 9 web + 6 UA-брендів + retrofit + ранні еталони), 4 виміри кожен.
- **copywriting/_RESEARCH_fedoriv_method.md** + **_ANTISLOP_ukrainian_hard.md** — метод Федоріва з джерел + жорсткий банлист (em-dash=0).
- Оновлені START_HERE, PIPELINE (Федорів у Ф1, motion+Higgsfield у Ф2, жорсткий anti-slop у Ф4), _INDEX.
