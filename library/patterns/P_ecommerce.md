# 🏗️ ПАТЕРН — E-COMMERCE / DTC-бренд (продукт-фокус)
> Тип: гібрид. Темп середній. Продукт-зірка, вигода, простота купівлі.

## КАРТА СЕКЦІЙ:
1. HERO — продукт + обіцянка + CTA «Купити/Дивитись». Крупне фото/відео продукту. → R_hero
2. БЕСТСЕЛЕРИ/категорії — сітка карток-продуктів, hover-reveal. → R_work (варіант картки)
3. ЦІННОСТІ/USP — чому ми (3-4 переваги, іконки).
4. ІСТОРІЯ бренду — олюднення (DTC). → R_about
5. ВІДГУКИ/UGC — соц-докази, фото клієнтів, counter (продано N).
6. ПРОДУКТ деталі — крупно, mask-reveal, складники/матеріали.
7. CTA — «Замовити». → R_cta
8. FOOTER — доставка, повернення, контакти.
## НОРМИ: 7-9 секцій. Чиста база + бренд-акцент. Швидкий доступ до купівлі. Реальні фото (не сток).
## ЕМОЦІЯ: бажання→довіра(відгуки)→легкість купівлі. Двошаровий копі: продукт(utility)+бренд(емоція).

## + ЗБАГАЧЕННЯ (з Beauty in Stem DTC):
- ПЕРЕВИЗНАЧЕННЯ КАТЕГОРІЇ продукту в hero (скінкер→«skin fitness») — свіжий кут продажу.
- 4-КРОКОВИЙ РИТУАЛ використання (дієслова: Cleanse→Activate→Recovery→Maintain) = продуктизація, кожен крок=продукт.
- ЦІНИ ОДРАЗУ в колекції (DTC — прибрати тертя купівлі).
- Опис продукту = ВИГОДА+РЕЗУЛЬТАТ, не склад («leaves skin brighter»).
- Науково-тех регістр для емоційного продукту = диференціація («engineered», «formulas»).
