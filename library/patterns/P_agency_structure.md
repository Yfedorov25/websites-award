# 🏗️ ПАТЕРН СТРУКТУРИ — Сайт агенції/студії (TIER 1)
> З замірів Cuberto/14islands/NK. Готова карта секцій для типу «агенція/послуга/портфоліо».

## КАРТА СЕКЦІЙ (згори вниз):
1. **HERO** (100vh або 0.7-0.8vh) — Big Idea 3-6 слів + підзаг + CTA. → R_hero
2. **INTRO/маніфест** (0.8-1vh) — 1-2 речення позиції («no cookie-cutter» Cuberto). Довіра.
3. **WORK/проєкти** (НАЙБІЛЬША, 3-6vh) — кейси з результатом. Hover-reveal карток. → R_work
4. **SERVICES** (2-3vh) — список послуг, stagger-поява. → R_services
5. **ABOUT/process** (1-2vh) — хто ми, цифри-докази.
6. **BLOG/insights** (опц., 1vh) — експертиза.
7. **CTA-фінал** (1vh) — великий заголовок + магнітна кнопка. → R_cta
8. **FOOTER** — контакти, marquee, великий лого.
## НОРМИ: 6-8 секцій, 10-15 екранів скролу. Work — серце (найбільша).
## ЕМОЦІЙНА ДУГА: інтрига(hero)→довіра(intro)→захват(work)→ясність(services)→людяність(about)→запал(CTA).

## + ВАРІАНТ: VIDEO-PRODUCTION / MOTION-АГЕНЦІЯ (з Podium — для Rozgon!):
hero(ніша-фокус) → showreel-портфоліо(картки: бренд+назва ролика+рік) → місія → behind-the-scenes → послуги(прод-ланцюг: Creative Direction/Production/Post/VFX) → клієнти(лого-стрічка) → [спортсмени/люди-докази] → CTA(метафора з домену).
- Типо: bold геометричний (Futura) для енергії, або під тон.
- Копі: НАСКРІЗНА метафора з домену клієнта (спорт→«finish line»/«step one»). Докази через бренди+імена.
- Відео-зірки: showreel-картки замість статичних кейсів.
- Застосовно: Rozgon, motion/відео/рілз-агенції.
