# 📘 ПЛЕЙБУК — КОЛІР/ПАЛІТРА award-сайту
## 1. РОЛЬ: палітра = емоція бренду + глибина. Колір несе настрій до того, як прочитають текст.
## 2. СИСТЕМА (токени з ролями, не випадкові):
--color-bg, --color-surface, --color-text-primary, --color-text-muted, --color-accent, --color-border, --color-overlay. Мінімум 6 ролей. ERA = 99 токенів (кожна кнопка×стан).
## 3. ПОБУДОВА: 1) обери базу (темна/світла з характером) → 2) 1-2 акценти → 3) alpha-шкала базового (.25/.4/.5/.6/.8 — NK) для глибини/оверлеїв → 4) ролі-токени → 5) перевір контраст ≥4.5:1.
## 4. ДЕРЕВО ПАЛІТРИ ПО НІШІ:
- Luxury/RE → темно-синій+тепла земля (ERA #051936+#c07863). Вечірнє, дороге.
- B2B/консалтинг → кремова/графіт (Lumena #f6f4e9). Інтелектуально.
- Gaming/bold → темний космос+alpha (NK #050311). Епічно.
- Креатив → контраст/біла або #0d0d0d. Або RGB на ч/б (Lusion).
- E-commerce → чиста+бренд-акцент. Продукт-фокус.
- Едтех → тепла доступна.
## 5. ПРАВИЛА: НЕ чистий #000/#fff (банально — бери #0d0d0d/#f6f4e9). Alpha-шкала замість сірих оверлеїв. Обмежена палітра (1 база+1-2 акценти). Градієнти дозовано (ERA blue→pink).
## 6. КОД:
```css
:root{
  --c-bg:#050311; --c-text:#fff; --c-muted:rgba(255,255,255,.6); --c-accent:#c07863;
  --c-black-25:rgba(5,3,17,.25); --c-black-50:rgba(5,3,17,.5); --c-black-80:rgba(5,3,17,.8);
}
```
## 7. ПОМИЛКИ: чистий #000/#fff · випадкові кольори без ролей · сірі оверлеї замість alpha · веселка без системи · слабкий контраст.
## 8. ПРИКЛАДИ: ERA 99 токенів темно-синій+теракота; NK alpha-шкала #050311; Lumena кремова #f6f4e9; Lusion RGB на ч/б.
## 9. МОБІЛЬНИЙ: ті самі токени, перевірити контраст на малих екранах/сонці.
## 10. ЧЕК: ☐ 6+ ролей-токенів ☐ alpha-шкала ☐ база з характером (не банальна) ☐ обмежена палітра ☐ контраст ≥4.5:1.
