# 📘 ПЛЕЙБУК — FOOTER (контакти+великий лого+marquee)
## 1. РОЛЬ: контакти, нав-дублікат, фінальний акорд бренду. Часто гігантський лого-текст.
## 2. ТИПИ: контакти+соцмережі+© · гігантський лого-текст на всю ширину · marquee-стрічка · CTA-повтор.
## 3. ПОБУДОВА: 1) контакти(email/адреса/соц) → 2) нав-дублікат → 3) гігантський лого-текст (обрізаний в'юпортом) → 4) ©рік+бренд → 5) parallax/reveal лого.
## 4. ДЕРЕВО: бренд→гігантський лого-текст; B2B→структуровані контакти+sitemap; креатив→marquee+великий лого.
## 5. НОРМИ: email+адреса+3-5 соц+нав+©. Лого-текст clamp(60px,18vw,300px). Висота 0.5-1vh.
## 6. КОД:
```js
gsap.to('.footer__big',{yPercent:-10,scrollTrigger:{trigger:'.footer',scrub:true}}) // parallax лого
// marquee: gsap.to('.track',{xPercent:-50,duration:20,ease:'none',repeat:-1})
```
## 7. ПОМИЛКИ: footer-смітник (все підряд) · нема фінального акорду · мертвий (нуль руху) · дрібний лого.
## 8. ПРИКЛАДИ: Lusion адреса+соц+©2026+«Built with ❤️»; Cuberto 2 офіси; гігантський лого-текст обрізаний знизу.
## 9. МОБІЛЬНИЙ: лого менше, контакти в стек.
## 10. ЧЕК: ☐ контакти повні ☐ фінальний акорд(лого) ☐ є рух ☐ нав-дублікат ☐ © актуальний.
