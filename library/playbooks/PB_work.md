# 📘 ПЛЕЙБУК — WORK/ПРОЄКТИ (серце портфоліо/агенції)
## 1. РОЛЬ: докази майстерності. Найбільша секція (Cuberto 6.2vh). Емоція: захват+довіра. Головна конверсійна для агенцій.
## 2. ТИПИ (обери):
| Тип | Вигляд | Коли | Еталон |
|---|---|---|---|
| Крупні картки вертикально | full/2-col великі | мало кейсів, кожен важливий | Cuberto, NK |
| Тег-список рядками | name+category, hover-прев'ю | багато кейсів | 14islands |
| Кейс-блок з відео+цифрами | опис+медіа+метрики | докази результату | Hildén&Kaira |
| Горизонтальний скрол | pin+x-scroll галерея | візуальні проєкти | E11 |
## 3. ПОБУДОВА: 1) обери тип за к-стю кейсів → 2) сітка (full/2-col/3-col) → 3) для кожного: медіа+назва+категорія+результат → 4) stagger-поява fade-up 0.1s крок → 5) hover-реакція картки → 6) мобільний 1 колонка.
## 4. ДЕРЕВО: ≤6 кейсів важливих→крупні картки; багато→тег-список; треба докази→кейс+цифри; візуальне→горизонт.скрол.
## 5. НОРМИ: 4-10 кейсів (лендинг 3-4). Назва 1-4 слова. Опис-результат 30-80 знаків. Тег 1-2 слова.
## 6. КОД:
```js
gsap.from('.work__item',{y:60,opacity:0,duration:1,ease:'power3.out',stagger:0.1,scrollTrigger:{trigger:'.work',start:'top 70%'}})
item.onmouseenter=()=>gsap.to(item.querySelector('img'),{scale:1.05,duration:0.6,ease:'power2.out'})
```
## 7. ПОМИЛКИ: опис=назва не результат · нема hover-реакції · усі картки однакові (нема ритму) · lorem-зображення · дрібні прев'ю.
## 8. ПРИКЛАДИ: Cuberto «Punto Pago–First Super-App LatAm» (результат); H&K кейс+104k views; 14islands тег-список name+ніша.
## 9. МОБІЛЬНИЙ: 1 колонка, hover→tap, зберегти співвідношення медіа.
## 10. ЧЕК: ☐ результат не назва ☐ hover-реакція ☐ stagger ☐ реальні медіа ☐ ритм карток ☐ мобільний стек.
