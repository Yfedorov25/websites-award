# 📘 ПЛЕЙБУК — СІТКА І ПРОСТІР (повітря = дорого)
## 1. РОЛЬ: простір і вирівнювання = «редакторськість». Багато повітря = дорого; щільність = дешево.
## 2. СИСТЕМА: 12-колонкова сітка + 8pt вертикальний ритм + clamp бічні поля.
## 3. ПОБУДОВА: 1) 12 колонок (Lusion calc((100%-11*2vw)/12)) → 2) max-width контенту 1200-1440 → 3) бічні clamp(20px,5vw,80px) → 4) padding секцій clamp(80px,12vh,200px) → 5) усі відступи кратні 8 → 6) навмисна асиметрія (зсув у сітці).
## 4. ПРАВИЛА: вертикальний ритм 8pt (8/16/24/32/48/64/96/128). Багато повітря між секціями. Контент вирівняний за сіткою, не «на око». Асиметрія = редакторськість (не все по центру).
## 5. КОД:
```css
:root{ --gap:2vw; --col:calc((100% - 11*var(--gap))/12); --pad-x:clamp(20px,5vw,80px); --pad-y:clamp(80px,12vh,200px); }
.section{padding:var(--pad-y) var(--pad-x)} .container{max-width:1440px;margin:0 auto}
.grid{display:grid;grid-template-columns:repeat(12,1fr);gap:var(--gap)}
```
## 6. ПОМИЛКИ: щільно тулити (нема повітря) · все по центру (нудно) · відступи не кратні 8 · контент прилипає до країв · нема max-width (розтягнуто на 4K).
## 7. ПРИКЛАДИ: Lusion 12-кол 2vw гаттери, padding clamp(30px,4vw,50px); award-сайти багато повітря між секціями.
## 8. МОБІЛЬНИЙ: 12→4 або 1 колонка, бічні clamp вниз до 16-20px, зберегти ритм.
## 9. ЧЕК: ☐ 12-кол сітка ☐ 8pt ритм ☐ багато повітря ☐ max-width ☐ бічні clamp ☐ асиметрія навмисна.
