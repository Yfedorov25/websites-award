# 📘 ПЛЕЙБУК — SERVICES/ПОСЛУГИ (що ми робимо)
## 1. РОЛЬ: ясність пропозиції. Емоція: впевненість+ясність. Часто продуктизація 3 пакети.
## 2. ТИПИ: сітка карток 3-кол · акордеон (клік розкриває) · 3 пакети-колонки з ціною · нумерований список 01-05 розгорнутий (Lumena/REF).
## 3. ПОБУДОВА: 1) формат за к-стю(3-7 послуг) → 2) нумерація 01/02 (моно, «системність») → 3) назва+опис+(ціна) → 4) stagger-поява → 5) hover/акордеон → 6) мобільний 1 кол.
## 4. ДЕРЕВО: 3 чіткі пропозиції→пакети-колонки; багато послуг→сітка/акордеон; B2B-глибокі→нумерований розгорнутий (Lumena 01-05 з підходом+результатом).
## 5. НОРМИ: 3-7 послуг АБО 3 пакети. Назва 1-3 слова. Опис 40-100 знаків.
## 6. КОД:
```js
gsap.from('.service',{y:40,opacity:0,duration:0.9,ease:'power3.out',stagger:0.08,scrollTrigger:{trigger:'.services',start:'top 75%'}})
// акордеон: gsap.to(panel,{height:'auto',duration:0.5,ease:'power2.inOut'})
```
## 7. ПОМИЛКИ: послуги без різниці · нема ієрархії · опис водою · нема нумерації (втрата «системності»).
## 8. ПРИКЛАДИ: REF дієслівні гасла капсом (DRIVE GROWTH); Lumena 01-05 розгорнуті з підходом+результатом; H&K Organic→Ads→Training.
## 9. МОБІЛЬНИЙ: 1 кол, акордеон зручний для тапу.
## 10. ЧЕК: ☐ нумерація ☐ чітка різниця послуг ☐ опис конкретний ☐ stagger ☐ мобільний стек.
