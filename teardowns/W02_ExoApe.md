# Exo Ape · Teardown #W2 — exoape.com (Layer 3) ✅
> WEB-блок #2. Нідерланди (глобально). Преміальна editorial-естетика. Hospitality/lifestyle.
> **Категорія:** WEB/brand/editorial. Проєкти: St. Regis Venice, Ottografie.

## СТЕК (probe)
- **Nuxt + GSAP, БЕЗ WebGL** (canvas=0; gsapVersions=true). Перший Nuxt у web-блоці. 9 скриптів. Lenis=false.
- Шрифт: **Lausanne** (300/400/500 ваги) — ОДНА родина (той самий преміум-гротеск що Garden Eight). Дисципліна. Біла база.

## АНАТОМІЯ РУХУ
- Мінімальний CSS-рух (0.5s×5, 0.2s×4, ease-in-out). Основний рух у GSAP (не CSS). Стриманий — рух не домінує, домінує контент (фото/відео).
- opacity-центричний (прості fade).

### Емоція:
Білий + один гротеск + мінімум руху + великі фото/відео = спокій, простір, «дихання». Editorial-розкіш через стриманість і якість контенту, не через ефекти. (Стратегія «стриманість» з нашої дихотомії.)

## АРХІТЕКТУРА / КОНТЕНТ
- «Work in motion» + «PlayReel» — відео-секції (поєднання web+video).
- Нав: Work / Studio / News / Contact. Класична чиста структура.
- Hospitality-фокус: «exceptional experiences where people **live, work and unwind**».

## СЕНСИ
- «where people live, work and unwind» — емоційно-просторова лексика (hospitality).
- Стриманий преміум: менше слів, більше простору.

## ЧОГО ВЧИМОСЯ / В БАЗУ
- **Editorial-розкіш через стриманість** (1 гротеск + білий + мінімум руху + великий медіа-контент) → niche-profiles (hospitality/lifestyle/luxury).
- **Lausanne** — 2-ге підтвердження (GE+ExoApe) → toolbox (преміальний гротеск).
- **Контент домінує над рухом** → philosophy (для медіа-важких ніш рух мінімальний, фото/відео — зірка).
- **Nuxt у web (не лише motion)** → стек.
