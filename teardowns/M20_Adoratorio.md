# Adoratorio Studio · Teardown #23 — adoratorio.studio (Layer 3 + MOTION) ✅
> Еталон #21 — ФІНАЛ motion-блоку. Брешіа+Амстердам. Independent design studio.
> **Категорія:** MOTION/WebGL + design.

## СТЕК (probe)
- **Vue + WebGL2 + GSAP** (gsapVersions=true). 1 canvas, **1 скрипт** (компактний бандл, як Aristide). Lenis=false.
- Шрифт: **PP Adoratorio** ЄДИНИЙ — ВЛАСНИЙ фірмовий (Pangram Pangram кастом). Максимальна айдентика (як LusionMono/LocomotiveNew). Світло-сіра база `#e6e6e6`.

## АНАТОМІЯ РУХУ (виміряно)
- Повільно-преміальний: **1.25s** (16×) + **1s** (17×) + 0.5s (18×). Європейський «спектакль»-темп.
- Easing: **`cubic-bezier(0.215,0.61,0.355,1)`** (56× — абсолютна домінанта!) = **easeOutCubic** (м'яке плавне гальмо, близько до дефолтного easeOutQuad).
- Властивості — МОРФІНГ ФОРМИ: `max-width, border-radius, transform` разом (16×) + `max-height` (16×). Анімують розміри/форму контейнерів, не лише позицію → блоки «розгортаються/морфують», не просто з'являються.

### Емоція:
1.25s + easeOutCubic + морфінг розмірів + сіра база + власний шрифт = плавне, цілісне, авторське «розгортання форми». Преміально-споглядальне.

## НОВЕ: морфінг розмірів як прийом
- Більшість студій анімують opacity+transform (поява). Adoratorio анімує **max-width/max-height/border-radius** — блоки змінюють ФОРМУ (розширюються, закруглюються). Це інший клас руху: не «прихід», а «трансформація на місці». Емоція: органічність, «живі» контейнери.

## ЧОГО ВЧИМОСЯ / В БАЗУ
- **easeOutCubic `cubic-bezier(0.215,0.61,0.355,1)`** → easing-бібліотека (поряд із easeOutQuad — обидва м'які easeOut, індустрія тяжіє сюди).
- **Морфінг форми** (max-width/height/border-radius) як окремий клас руху → motion-system (трансформація на місці vs поява).
- **Власний шрифт** (PP Adoratorio) — 4-й приклад → patterns айдентики (власний шрифт = фірмова деталь топ-студій).
- **1 скрипт компактний бандл** (як Aristide) → toolbox.
