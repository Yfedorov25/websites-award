# Order · Teardown #W6 — order.design (Layer 3) ✅
> WEB-блок #6. Бруклін+Маямі. Brand identity office (ex-Pentagram: Jesse Reed, Hamish Smyth). Standards Manual.
> **Категорія:** WEB/brand/editorial (чистий брендинг). Клієнти: Herman Miller, Kickstarter, NYU.

## СТЕК (probe) — НАЙПРОСТІШИЙ З УСІХ ★
- **Vanilla, БЕЗ WebGL, БЕЗ фреймворку, БЕЗ GSAP** (всі false). 7 скриптів. Чистий CSS-рух.
- Доказ: топ brand-студія свідомо НЕ потребує жодних модних бібліотек. Лише HTML/CSS/трохи JS + сильна типографіка/контент. Протилежний полюс до Lusion (власний WebGL-движок).

## ТИПОГРАФІКА — 11 ШРИФТІВ (протилежність дисципліні) ★
- Pitch Sans, Pastiche Grotesque, Plebeian, Etude, Spades, Stringer, Vanity, Haltung, Sita Serif, Sita Sans + OrderArrow (власний для стрілок).
- Це БРЕНДИНГ-студія → демонструє типографічний діапазон, кожен кейс своїм шрифтом. Свідома протилежність «одному гротеску». Контекст: коли студія сама ПРО типографіку — багато шрифтів = демо майстерності.

## ПАЛІТРА
- Темна `#141414`. Мінімалізм.

## АНАТОМІЯ РУХУ
- Простий CSS: `0.75s, 0.5s` (18× — пара) + 0.25s (19×). Easing: переважно дефолтний **`ease`** (!), не кастомні криві.
- Властивість: **`filter + opacity`** (18×) — поява з blur/яскравістю (blur→sharp reveal). + opacity (21×).
- Найпростіший рух з усіх: дефолтний ease, без GSAP. Доводить — рух не мусить бути складним.

## СЕНСИ / КОПІРАЙТ — антидекоративний маніфест ★
- «We do everything **for a reason, never for decoration**.» — філософія стриманості як ідентичність.
- Це пояснює простий стек: вони ПРАКТИКУЮТЬ те, що проповідують — нуль декоративних ефектів, лише функція+сенс.

## ЧОГО ВЧИМОСЯ / В БАЗУ
- **Award brand-студія БЕЗ WebGL/GSAP/фреймворку** → philosophy (найсильніший доказ: концепція>технологія; стриманість може бути позицією).
- **«For a reason, never for decoration»** → copywriting + philosophy (антидекоративна стратегія).
- **Багато шрифтів коли студія ПРО типографіку** (контекст-залежно) → toolbox (дисципліна 1 шрифту НЕ універсальна — залежить від позиціонування).
- **filter+opacity (blur→sharp) reveal** на дефолтному ease → motion-system (простий ефективний прийом).
- **OrderArrow** (власний шрифт лише для стрілок/іконок) → patterns (іконки як шрифт).
