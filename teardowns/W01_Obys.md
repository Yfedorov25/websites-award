# Obys Agency · Teardown #W1 — obys.agency (Layer 3) ✅
> WEB-блок #1. Awwwards Studio of the Year 2023, CSSDA Studio of the Year 4×. Концепт-драйвена студія, EU/Україна.
> **Категорія:** WEB/brand/editorial. Клієнти: CNN, Porsche (Taycan), Hilton, Miro, Glyphic Biotech, Peter Lindbergh.

## СТЕК (probe)
- **Vanilla + WebGL2 + GSAP** (рух повністю на JS/GSAP, НЕ CSS-transitions — CSS-probe порожній). canvas=1, 2 скрипти (дуже компактно). Lenis у бандлі/GSAP ScrollSmoother.
- Шрифти: **Obys + ObysSans4** — ДВА ВЛАСНІ шрифти (повна кастомна типо-родина!). Найвищий рівень айдентики досі (не один власний, а родина). Біла база.

## ТИПОГРАФІКА — максимальна власна айдентика ★
- Два власні шрифти = студія має ВЛАСНУ типосистему як продукт. Це фінальний рівень: Lusion/Locomotive/Adoratorio мали 1 власний шрифт, Obys — родину. Сигнал: «наш дизайн настільки самобутній, що навіть букви наші».

## РУХ
- Повністю GSAP-driven (CSS-transitions майже не використовують). Рух програмний, не декларативний → повний контроль таймлайнів. Для нас: складніший, але гнучкіший підхід (GSAP timelines замість CSS).

## АРХІТЕКТУРА / КОНТЕНТ
- Портфоліо = головний контент: **62+ проєктів** із чіткою тег-системою: `[Категорія клієнта] + [Послуги: Concept / Web Design+Dev / Identity / 3D / Creative Direction]`.
- Мінімальна нав: Work / About (тільки суть).
- Тег-система послуг = прозоро показує що вміють (Concept→Identity→Web→3D повний цикл).

## СЕНСИ / КОПІРАЙТ
- «**Concept-driven** design studio... award-winning brand and web experiences shaped by **storytelling and strong visual systems**».
- Якорі: storytelling + visual systems + concept. Позиція: не виконавці, а концептуалісти.

## ЧОГО ВЧИМОСЯ / В БАЗУ
- **Власна типо-родина** (2 шрифти) = вищий рівень айдентики → patterns (для нашого бренду — хоча б 1 кастомний акцидентний шрифт).
- **GSAP-driven рух** (timelines, не CSS) для повного контролю → toolbox (коли потрібна складна оркестрація).
- **Тег-система портфоліо** `[ніша]+[послуги]` → patterns (прозорий показ компетенцій).
- **«Concept-driven» позиція** → copywriting (продавати концепцію, не виконання).
- **Компактність** (2 скрипти на award-сайт) → philosophy (мінімум залежностей).
