# Tendril · Teardown #12 — tendril.studio (Layer 3 + MOTION ANATOMY) ✅
> Еталон #10. Торонто+Сан-Паулу. CGI/design/animation студія. Клієнти: Louis Vuitton×F1, FX, IBM, Figure, Mercedes.
> **Категорія:** 3D-motion/CGI (для ВІДЕО, не веб-WebGL). Перший приклад де дістав точні цифри руху.

## СТЕК (probe)
- **НЕ WebGL-сайт** (canvas=0). Сайт-портфоліо: контент = відео-роботи (CGI рендериться офлайн, на сайт іде як відео). Сам сайт легкий.
- Vanilla, без фреймворку. 10 скриптів. Lenis=false. GSAP не на window (можливо в бандлі або чистий CSS/RAF).
- 1 шрифт: **HK Grotesque** (чистий нейтральний гротеск). Біла база. Мінімалізм — щоб не відволікати від відео-робіт.

## АНАТОМІЯ РУХУ (виміряно — еталонний приклад) ★★★
- **Домінантний entrance: `clip-path + opacity` разом** (24× — найчастіший патерн на сайті).
  - Тривалість: **1s** (повільно, кінематографічно).
  - Easing: **`cubic-bezier(0.65, 0, 0.35, 1)`** = **easeInOutCubic** (симетрична S-крива: плавний старт + плавне гальмо).
  - Ефект: елементи **розкриваються шторкою (clip-path)** з одночасним проявленням. Відчуття «завіса піднімається» — театральне, преміальне, вагоме розкриття.
- Вторинні переходи:
  - `opacity + transform` (6×, 0.6s) — fade зі зсувом.
  - `color` (9×, 0.4s) — hover на лінках/тексті.
  - `0.2s` (4×) — мікровзаємодії (миттєвий відгук кнопок).
- **Триярусний темп:** 1s (драматичні розкриття контенту) / 0.4-0.6s (UI-переходи) / 0.2s (мікро-відгук). Кожен ярус — своя роль.

### Емоція цього руху:
Повільні (1s) + симетрична S-крива + clip-path-шторка = **«дорогий», вагомий, кінематографічний** рух. Доречно для CGI-студії: сам рух сайту має відповідати якості їхнього відео — нічого різкого/дешевого. Кожен елемент «коштує» свою секунду розкриття.

## ЧОГО ВЧИМОСЯ / В БАЗУ
- **clip-path + opacity reveal @1s easeInOutCubic** — конкретний рецепт «театрального розкриття» → emotion-mechanics + toolbox (готовий до повторення).
- **Триярусний темп руху** (1s контент / 0.4s UI / 0.2s мікро) → motion-system (різні швидкості для різних ролей).
- **easeInOutCubic `cubic-bezier(0.65,0,0.35,1)`** — симетрична S для преміального/вагомого → easing-бібліотека.
- **Мінімалізм сайту коли продукт = відео** → philosophy (сайт не має конкурувати з контентом).
- **Наративний копірайт по проєктах** (кожен кейс — історія з емоційним дієсловом) → copywriting.
- **Рух сайту = рівень продукту** (CGI-студія = вагомий рух; не можна дешевий) → philosophy.
