# Use All Five · Teardown #22 — useallfive.com (Layer 3 + MOTION) ✅
> Еталон #20. LA. UI/UX design+technology. Клієнти: Google (10+ років — Arts&Culture, News Initiative), UCLA.
> **Категорія:** UI/UX + brand (БЕЗ WebGL).

## СТЕК (probe)
- **React + Next, canvas=0 (БЕЗ WebGL).** UI/UX-агенція, рух на DOM/CSS/GSAP. 15 скриптів. Третій Next.
- Шрифти: **Bau Pro (+Medium)** (Bauhaus-гротеск) + **Fugue** (антиква). Біла база.

## АНАТОМІЯ РУХУ (виміряно)
- Швидко-середній: 0.25s (21×), 0.7s (15×), 0.2s (12×), 0.5s (5×). Двошарово: UI 0.2-0.25s + контент 0.5-0.7s.
- Easing: `ease-in-out` (11×) + `cubic-bezier(0.2,0.485,0.205,1)` (кастомний easeOut). transform домінує (12×) + height(6×) — анімують і розміри (акордеони/розкриття).

### Емоція:
Чистий UI/UX-рух: швидкий, функціональний, без драми. Біла база + Bauhaus-гротеск = раціональність, ясність (доречно для design-systems агенції).

## ЧОГО ВЧИМОСЯ / В БАЗУ
- **Друга UI/UX-агенція без WebGL** (після Cuberto) → philosophy (UI/UX-ніша не потребує 3D, рух функціональний).
- **Анімація height/розмірів** (не лише transform/opacity) → motion-system (розкриття/акордеони).
- **Bau Pro** → toolbox типосистем (Bauhaus-гротеск).
