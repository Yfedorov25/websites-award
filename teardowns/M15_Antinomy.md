# Antinomy Studio · Teardown #18 — antinomy.studio (Layer 3 + MOTION) ✅
> Еталон #16. Амстердам/Берлін/NY. «Global creative studio shaping contemporary brands.» Клієнти: Apple, Adidas, SpaceX, A24.
> **Категорія:** MOTION/WebGL + brand.

## СТЕК (probe)
- **Nuxt + WebGL2 + GSAP** (gsapVersions=true). **ЧЕТВЕРТИЙ Nuxt** (IG/Noomo/GE/Antinomy) — Nuxt домінує серед WebGL-студій. 1 canvas, 14 скриптів. Lenis=false.
- Шрифт: **ABC Diatype** ЄДИНИЙ (Dinamo — модний нейтральний гротеск). Один шрифт = дисципліна (як Telha). Біла база.

## АНАТОМІЯ РУХУ (виміряно)
- Середній темп: 0.5s (8×), 0.4s (6×), 0.2s (4×) + пара **`0.3s,0.6s`** (5× — розшарований stagger, як Cuberto).
- Easing: **`cubic-bezier(0.19,1,0.22,1)`** (easeOutExpo, як Cuberto) часто в парі з **linear** (комбо: одна властивість лінійно, інша expo).
- opacity домінує (16×) — fade-центричний, прості чисті появи.

### Емоція:
Помірний expo + білий + один гротеск + fade = «clarity, character» (зі слогану). Чисто, дисципліновано, без надмірності. Бренд-орієнтований спокій.

## СЕНСИ
- «brand experiences with **clarity, character, and lasting relevance**» — тріада цінностей (clarity/character/relevance).
- «shaping contemporary brands» — позиція бренд-формувача.

## ЧОГО ВЧИМОСЯ / В БАЗУ
- **Nuxt 4-те підтвердження** → стек-консенсус (Nuxt/Next+WebGL — індустрійний стандарт).
- **Один гротеск (ABC Diatype) + біла база + expo-fade** = дисциплінований brand-профіль → niche (brand studio).
- **Комбо linear+expo на різних властивостях** одного переходу → motion-system.
- **ABC Diatype** → toolbox типосистем (модний нейтральний гротеск 2020-х).
