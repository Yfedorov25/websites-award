# Sense Bank · UA-teardown #5 — sensebank.ua (повний метод) ✅
> Fintech, рестайл після націоналізації (екс-Альфа). Архетип Ruler+Everyman. GSAP, без WebGL.

## СТЕК
- GSAP, без WebGL. Шрифт **Mazzard** (геом. гротеск). Біла база. Простий рух 0.3s ease.

## КОПІРАЙТ / СЕНСИ ★ (гра з назвою бренду)
- **Обігрування семантики назви:** «**Є сенс** створювати світло разом» (Sense -> «є сенс»). Назва вшита в меседж = впізнаваність + сенс.
- **Патріотичний підтекст:** «світло» (енергонезалежність після обстрілів), «Національний кешбек», «системно-важливий український банк» (статус+гордість).
- CTA-дієслова: «Дивись», «Слідкуй».
- «техніку майбутнього» (та сама лексика майбутнього, що НП).
- Тон: впевнений, національно-свідомий, трохи статусніший за mono (Ruler-нота).

## У БАЗУ:
- **Обігрування семантики назви бренду в слогані** (Sense->сенс) — потужний прийом для брендів зі «значущою» назвою. Шукати подвійний сенс у назві клієнта.
- **Патріотичний шар через метафору** («світло» = незламність), не прямо. Тонше за прямі заклики.
- UA-fintech-патерн: біла база + геом. гротеск + простий рух + копірайт-флекс. Рух не головний, копі+меседж головні.
