# Gretel · Teardown #W7 — gretelny.com (Layer 3) ✅
> WEB-блок #7. Бруклін NYC. Незалежна branding/strategy студія. Клієнти: Nike, Vice, MoMA, Netflix, IBM.
> **Категорія:** WEB/brand/editorial. ШОК-ІНСАЙТ: на Squarespace.

## СТЕК (probe) — НАЙВАЖЛИВІШИЙ ІНСАЙТ ★★★
- **SQUARESPACE!** Доказ: шрифти `squarespace-ui-font` + `social-icon-font` (системні SS). 23 скрипти (типове SS-навантаження). Без WebGL/фреймворку/GSAP/Lenis.
- **Топ brand-студія (Nike/Netflix/MoMA/IBM) на NO-CODE конструкторі.** Це найсильніший доказ усього датасету: **дизайн/контент/типографіка/репутація >> технологія сайту.** Award-клієнти не дивляться на стек — дивляться на роботи.
- Контекст: для PURE brand-студії сайт = вітрина кейсів. Squarespace достатній, бо цінність у роботах, не в сайті-як-атракціоні. (Протилежність Lusion, де сайт САМ = демо можливостей.)

## ТИПОГРАФІКА
- **Chivo + DIN 2014 + Akkurat** (преміальні гротески) + системні SS. Біла база. Bold editorial (їхній фах — сміливі айдентики).

## АНАТОМІЯ РУХУ
- Змішаний: 0.3s (10×), 1.2s (5×), 0.8s пари (5×), 0.14s (7×). Easing: `cubic-bezier(0.33,0,0.2,1)` (8×) + Material (0.4,0,0.2,1) + linear. opacity-центричний (19×). Стандартний SS+кастомний CSS рух.

## СЕНСИ
- «independent branding, strategy and design studio» — просто, без пафосу. Робота говорить сама.

## ЧОГО ВЧИМОСЯ / В БАЗУ ★
- **СТЕК НЕ ВИЗНАЧАЄ РІВЕНЬ** (Gretel=Squarespace, клієнти Nike/Netflix) → philosophy (найсильніший доказ: для brand-вітрини важливі роботи/типографіка, не технологія). Звільняє від технологічного снобізму.
- **Для PURE brand-студії** сайт-вітрина може бути простим (SS/Webflow) — інвестувати в роботи, не в сайт-атракціон → business/philosophy.
- **АЛЕ:** це працює бо в них РЕПУТАЦІЯ. Для нас (будуємо ім'я) сайт-атракціон потрібен як демо — ми ще не Gretel. Нюанс: стратегія залежить від етапу.
- Контраст Lusion (сайт=демо движка) vs Gretel (сайт=проста вітрина): обидва award, бо обидва відповідають своєму позиціонуванню.
