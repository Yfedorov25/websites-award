# North Kingdom · Teardown #19 — northkingdom.com (Layer 3 + MOTION) ✅
> Еталон #17. Швеція. «Creative agency for the world of gaming.» Клієнти: EA, Riot, Supercell, 10 Chambers.
> **Категорія:** MOTION/WebGL + experience design. Ніша: ГЕЙМІНГ.

## СТЕК (probe)
- **React + Next + Lenis + WebGL2** — ДРУГИЙ Next+Lenis (після 14islands). Підтверджує наш цільовий стек. 1 canvas, 14 скриптів.
- Шрифти: повна родина **FK** (Florian Karsten): FK Grotesk Neue (Reg/Bold) + FK Display + FK Grotesk Mono. Одна типо-родина у 3 ролях (текст/дисплей/моно) = цілісність.
- База: темна `#050311` (синювато-чорна — «ігровий космос»).

## АНАТОМІЯ РУХУ (виміряно)
- Домінанта **0.6s** (50×) + 0.2s (22×) + 0.3s (17×) + 0.4s (7×). Середньо-швидкий «ігровий» темп.
- Easing: **`cubic-bezier(0.645,0.045,0.355,1)`** = easeInOutCubic (симетрична S, як Tendril).
- Багато парних тривалостей (0.3s,0.3s / 0.2s,0.2s) — синхронні переходи.

### Емоція:
0.6s + симетрична S + темна синя база = динамічно, але контрольовано — ігрова енергія без хаосу. Темна база + неонові акценти (типове для гейм-естетики) = занурення, «світ гри».

## НІША — ГЕЙМІНГ (нова)
- «creative agency for the world of gaming» — вузька спеціалізація. «connect players and brands through unforgettable experiences».
- Гейм-ніша = темна база + повна типо-родина (display для драми) + середньо-швидкий рух + WebGL-світи. Емоція: занурення, гра, енергія.

## ЧОГО ВЧИМОСЯ / В БАЗУ
- **Next+Lenis 2-ге підтвердження** → наш стек надійний.
- **Одна типо-родина в 3 ролях** (FK Grotesk/Display/Mono) = цілісність без зоопарку шрифтів → toolbox.
- **Гейм-ніша профіль** (темна синя база + display-драма + 0.6s рух) → niche-profiles (нова: gaming).
- **easeInOutCubic 2-ге підтвердження** (Tendril+NK) → симетрична S = популярна для «контрольованої драми».
