# Garden Eight · Teardown #15 — garden-eight.com (Layer 3 + MOTION) ✅
> Еталон #13. Токіо. Японська цифрова дизайн-студія. Перший із ВІДКРИТИМИ версіями бібліотек.
> **Категорія:** MOTION/WebGL. Артистична японська школа.

## СТЕК (probe) — версії видно! ★
- **GSAP 3.12.1 + Three.js r133 + Nuxt + WebGL2** — усі на window (не bundled, видно версії).
- **ТРЕТІЙ Nuxt-сайт** (IG, Noomo, GE) → Nuxt = очевидний фаворит топ-WebGL-студій. (Наш аналог Next+R3F валідний.)
- GSAP 3.12 = сучасний (не legacy як Resn). Three r133 = 2021, свіжіший за Resn r84.
- 1 canvas, 8 скриптів. Lenis=false (GSAP ScrollSmoother або власний).
- Шрифти: **gunsan** (дисплейний/унікальний) + **Lausanne** (преміальний швейц. гротеск, Weltkern). Прозора/світла база.

## АНАТОМІЯ РУХУ (виміряно) — ЯПОНСЬКА ШКОЛА ★★★
- **Темп ШВИДШИЙ за європейські студії:** домінанта **0.6s** (20×) + **0.3s** (18×) + 0.2s (4×). Майже НЕМАЄ 1.2s (лише поодинокі 1.4s).
- Контраст: Cuberto/Dogstudio = 1.2s домінанта (повільно-драматично). Garden Eight = 0.3-0.6s (швидко-точно).
- Easing: **`cubic-bezier(0.32, 0.94, 0.6, 1)`** (28×) — easeOut з раннім підйомом: швидкий старт, акуратне (не драматичне) гальмо. М'якший і стриманіший за expo.
- Властивості: opacity(19×)+transform(16×) — fade+зсув, простіше за clip-path Tendril.

### Емоція (японська естетика руху):
Швидко (0.3-0.6s) + стримана крива + простий fade-зсув = **точність, легкість, делікатність, «нічого зайвого»**. Рух не привертає увагу до себе — він обслуговує контент. Контраст до європейського «дорогого повільного спектаклю». Це wabi-sabi-підхід: стриманість як вишуканість.

## ШКОЛИ РУХУ (нове розуміння):
- **Європейсько-американська** (Cuberto/Dogstudio/Tendril): повільно (1-1.2s), драматично, рух = подія.
- **Японська** (Garden Eight): швидко (0.3-0.6s), стримано, рух = служба контенту.
→ Темп руху культурно/нішево обумовлений. Для нас: вибирати темп під бажану емоцію, не за звичкою.

## ЧОГО ВЧИМОСЯ / В БАЗУ
- **Японський швидкий-стриманий темп** (0.3-0.6s) як альтернатива європейському повільному → motion-system (темп = вибір під емоцію/культуру).
- **easeOut cubic-bezier(0.32,0.94,0.6,1)** — стримана крива → easing-бібліотека.
- **GSAP 3.12 + Three r133 + Nuxt** — сучасний валідний стек, версії підтверджені → toolbox.
- **Lausanne** → toolbox типосистем.
- **Nuxt 3-тє підтвердження** → стек-консенсус.
