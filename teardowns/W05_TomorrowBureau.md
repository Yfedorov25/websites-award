# Tomorrow Bureau · Teardown #W5 — tomorrowbureau.io (Layer 3) ✅
> WEB-блок #5. UK. «Speculative. Digital. Realities.» Засн. 2019. Клієнти: Adidas, HOKA, ON Running, Dior, Renault.
> **Категорія:** WEB/brand/editorial (art+design).

## СТЕК (probe)
- **React + Next** (canvas=1, context=null — 2D-canvas для ефектів). 13 скриптів. Lenis bundled, GSAP-driven рух.
- Шрифт: **Px Grotesk** ЄДИНИЙ (Optimo — швейц. гротеск). Дисципліна одного шрифту (як Antinomy/Bürocratik/ExoApe). Прозора база.

## АНАТОМІЯ РУХУ
- CSS мінімальний (0.2s×4) — рух у GSAP. opacity-центричний.

## СЕНСИ / КОПІРАЙТ
- Слоган-маніфест: «**Speculative. Digital. Realities.**» (тріада через крапки — ритм/вагомість кожного слова).
- «creative studio built on **ideas**», «conceptual depth and meticulous craft», «singular, high-impact visual narratives».
- Якорі: ideas / conceptual depth / craft (концептуалізм, як Obys).

## ЧОГО ВЧИМОСЯ / В БАЗУ
- **Тріада через крапки** як слоган-патерн («X. Y. Z.» — кожне слово вагоме) → copywriting.
- **Один гротеск (Px Grotesk)** — 5-те підтвердження дисципліни одного шрифту в brand-студіях → toolbox/philosophy.
- Патерн web-brand: Next + 1 гротеск + GSAP + мінімум CSS + великий медіа-контент (повторюється: ExoApe/Antinomy/TB).
